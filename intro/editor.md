# 编辑器

使用具备Dart插件的编辑器，能让开发事半功倍，以下是笔者推荐的编辑器/IDE。

## Jetbrains家族

[Jetbrains](https://www.jetbrains.com/)公司的IntelliJ IDEA、PhpStorm、WebStorm等，可以安装Jetbrains的[Dart](https://plugins.jetbrains.com/plugin/6351-dart)插件

## VS Code

来自微软Visual Studio团队的[VS Code](https://code.visualstudio.com/)，可以使用[Dart Code](https://marketplace.visualstudio.com/items?itemName=DanTup.dart-code)插件

## Atom

Github出品的[Atom](https://atom.io/)编辑器，可以使用[dartlang](https://github.com/dart-atom/dartlang/)插件

## Vim

老牌编辑器[Vim](http://www.vim.org/)，可以使用[dart-vim-plugin](https://github.com/dart-lang/dart-vim-plugin)插件

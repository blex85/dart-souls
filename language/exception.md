# 流程控制与异常处理

本节简述Dart的流程控制语句和异常处理。

## 流程控制语句

跟大部分编程语言类似，Dart具有以下流程控制语句

* if 、esle
* for 循环
* while、do-while 循环
* break、continue
* switch、case

## 异常处理

异常处理涉及两个类：`Exception` 和 `Error`。

### Exception

Exception类表示程序执行过程中的异常，它们如果没有被捕获，应用通常会挂起并中止执行。

# 运算符和流程控制

本节简述Dart的运算符和流程控制语句。

| :--- | :--- |

| 乘法运算符 | \*   /   %   ~/ |
| 加法运算符 | +   - |
| 移位运算符 | &lt;&lt;   &gt;&gt; |

| 按位异或 | ^ |
| 按位或 | \| |
| 关系与类型检测 | &gt;=   &gt;   &lt;=   &lt;   as   is   is! |
| 逻辑与 | && |
| 逻辑或 | \|\| |
| 是否为null | ?? |
| 条件运算符 | ? :  |
| 级联运算符 | .. |
| 赋值运算符 | = \*= /= ~/= %= += -= &lt;&lt;= &gt;&gt;= &= ^= \|= ??= |

## 流程控制语句

* if 、esle
* for 循环
* while、do-while 循环
* break、continue
* switch、case

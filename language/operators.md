# 运算符和流程控制

本节简述Dart的运算符和流程控制语句。

| 描述 | 运算符 |
| :--- | :--- |
| 一元运算符（后置） | ++   --   \( \)   \[ \]   .   _** ?.**_ |
| 一元运算符（前置） | -   !   ~   ++   -- |
| 乘法运算符 | \*   /   %   **~/** |
| 加法运算符 | +   - |
| 移位运算符 | &lt;&lt;   &gt;&gt; |
| 按位与 | & |
| 按位异或 | ^ |
| 按位或 | \| |
| 关系与类型检测 | &gt;=   &gt;   &lt;=   &lt;   _**as   is   is!**_ |
| 逻辑与 | && |
| 逻辑或 | \|\| |
| 是否为null | _**??**_ |
| 条件运算符 | ? : |
| 级联运算符 | _**..**_ |
| 赋值运算符 | = \*= /= ~/= %= += -= &lt;&lt;= &gt;&gt;= &= ^= \|= _**??=**_ |



与其他类C语言比较，Dart的运算符大同小异，这里选出部分Dart特有的运算符（即上表中使用斜粗体的运算符）进行讲解:

| 运算符 | 介绍 | 示例 |

| is  is! | 检查对象是否属于某种类型 | if \(coder is Programmer\) { } |
| as | 类型转换，如果转换失败将抛出异常 | \(coder as Programmer\).name |

## 流程控制语句

* if 、esle
* for 循环
* while、do-while 循环
* break、continue
* switch、case
